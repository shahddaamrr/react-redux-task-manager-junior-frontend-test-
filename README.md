# Frontend Coding Test

This repository contains two projects:

1. React Task Manager
2. React Native User List

---

## 1. React Task Manager

A simple task management web application built with React and Redux Toolkit.

### Features

- Add tasks
- Edit tasks
- Delete tasks
- Mark tasks as completed
- Filter tasks by priority
- Save tasks in localStorage
- Manage state using Redux Toolkit

### Run the project

1. Open the react-task-manager folder.
2. Run npm install.
3. Run npm run dev.
4. Open the local development link in the browser.

---

## 2. React Native User List

A React Native Expo application that displays users from an API.

### Features

- Fetch users from JSONPlaceholder API.
- Store users in Redux.
- Cache users using AsyncStorage.
- Display users using FlatList.
- Search users by name.
- Load more users.
- Display user data in a reusable UserCard component.
- Format address as street, city, and zipcode.

### Run the project

1. Open the react-native-user-list folder.
2. Run npm install.
3. Run npx expo start.
4. Open the app using Expo Go or the web browser.
